# Zakrajshek.com

A modern, responsive personal-brand website for William Zakrajshek.

## Deploy on GitHub Pages

1. Upload all files in this folder to the root of `3dudes1life/zakrajshek.github.io`.
2. In GitHub, open **Settings → Pages**.
3. Set the source to **Deploy from a branch** and choose `main` / root.
4. The site will publish at `https://3dudes1life.github.io/zakrajshek.github.io/` until the custom domain is connected.

## Connect Zakrajshek.com

The included `CNAME` file points GitHub Pages to `zakrajshek.com`. Configure the domain's DNS records with your domain registrar according to GitHub Pages' current custom-domain instructions.

## Add William's Photo

Place a portrait at:

`assets/william.jpg`

Recommended: vertical crop, 1600 × 2000 px or larger. The placeholder automatically disappears when the image is present.

## Easy Edits

- Main content: `index.html`
- Colors and layout: `assets/styles.css`
- Mobile menu and reveal effects: `assets/script.js`
